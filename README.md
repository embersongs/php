# PHP для WEB511

**Курс «Создание web–приложений, исполняемых на стороне сервера при помощи языка программирования PHP и СУБД MySQL
»**  
*Академия ТОП | Группа WEB511*

## 📌 О курсе

Этот репозиторий содержит все учебные материалы, примеры кода и домашние задания по курсу PHP для группы **WEB511**.

В рамках курса мы изучаем серверную веб-разработку: от основ синтаксиса до работы с базами данных и создания динамических сайтов.

## 🎯 Цели курса

- Освоить синтаксис и возможности PHP 8.x
- Научиться работать с формами, сессиями и cookies
- Подключиться к базе данных и выполнять CRUD-операции
- Понять MVC (модель-представление-контроллер)
- Познакомиться с фреймворком Laravel
- Создать полноценный веб-проект (блог)
